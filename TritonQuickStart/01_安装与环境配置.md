## 1.1 什么是 Triton-Ascend

Triton-Ascend 是 [OpenAI Triton](https://github.com/triton-lang/triton) 在华为昇腾 NPU 上的适配版本。它保留了标准 Triton 的编程模型和 API，使开发者可以用相同的 Triton 代码在 NPU 上编写高性能算子内核。

### 核心特性

- **标准 Triton API**: 使用 `@triton.jit`、`tl.load`、`tl.store`、`tl.dot` 等熟悉的 API
- **NPU 后端**: 将 Triton IR 编译为昇腾 CANN 可执行的指令
- **自动调优**: `@triton.autotune` 自动搜索最优 BLOCK 配置
- **多核并行**: 利用 NPU 的 AI Core 实现数据并行

### 昇腾 NPU 架构 (Ascend 910B)

910B 的 AI Core 中，AIC（Cube Core，矩阵计算）和 AIV（Vector Core，向量计算）是**相互隔离**的，各自拥有独立的片上存储。L2 Cache 为 AIC 和 AIV 共享。

```
AI Core 结构 (Ascend 910B):

                        +-----------------------+
                        |   Global Memory (HBM)  |
                        +-----------------------+
                                    |
                                    v
                        +-----------------------+
                        |   L2 Cache (共享)      |  <--- AIC 与 AIV 共用
                        +-----------------------+
                           /                \
                          /                  \
                         v                    v
   +--------------------------------+   +----------------------------------+
   |     AIC (Cube Core)            |   |     AIV (Vector Core)            |
   |     矩阵乘法 (tl.dot)          |   |     向量计算 (tl.load/store/sum) |
   |                                |   |                                  |
   |   +--------+ +--------+       |   |   +----------------------------+ |
   |   |  L0A   | |  L0B   |       |   |   |  Unified Buffer (UB)       | |
   |   |(左矩阵)| |(右矩阵)|       |   |   |  ~192KB / A2: ~1MB         | |
   |   +--------+ +--------+       |   |   |  tl.load / tl.store        | |
   |        |          |            |   |   |  tl.sum / tl.max / tl.exp  | |
   |        v          v            |   |   +----------------------------+ |
   |   +-----------------+          |   +----------------------------------+
   |   |      L0C        |          |
   |   |   (累加器)       |          |
   |   +-----------------+          |
   |        ^          |            |
   |        |          v            |
   |   +-----------------+          |
   |   |  L1 Buffer      |          |
   |   |  (~1MB)         |          |
   |   +-----------------+          |
   +--------------------------------+

数据流:
  GM --> L2 --> AIC.L1 --> AIC.L0A/L0B --> (tl.dot) --> AIC.L0C --> AIC.L1 --> L2 --> GM
  GM --> L2 --> AIV.UB  --> (向量计算)   --> AIV.UB  --> L2 --> GM
```

## 1.2 环境要求

### 硬件要求

| 项目 | 要求 |
|------|------|
| NPU | Ascend 910A / 910B 系列 |
| CPU | x86_64 或 AArch64 |
| 内存 | >= 32GB |

### 软件要求

| 软件 | 版本要求 |
|------|---------|
| Python | 3.9 - 3.11 |
| CANN | 8.5.0（推荐） |
| PyTorch | 2.7.1 |
| torch_npu | 2.7.1 |

---

## 1.3 安装步骤

### 第一步：安装 CANN

从[昇腾社区](https://www.hiascend.com/cann/download)下载并安装 CANN 8.5.0。

```bash
# 配置 CANN 环境变量
source /usr/local/Ascend/ascend-toolkit/set_env.sh
# 或非 root 用户：
source ${HOME}/Ascend/ascend-toolkit/set_env.sh
```

### 第二步：安装 PyTorch 和 torch_npu

```bash
# 安装 PyTorch
pip install torch==2.7.1+cpu --index-url https://download.pytorch.org/whl/cpu

# 安装 torch_npu（昇腾 PyTorch 适配层）
pip install torch_npu==2.7.1
```

### 第三步：安装 Triton-Ascend

**方式一：pip 安装（推荐）**

```bash
pip install triton-ascend
```

**方式二：源码安装**

```bash
git clone https://gitee.com/ascend/triton-ascend.git
cd triton-ascend
pip install -e .
```

**方式三：Docker 安装**

```bash
docker pull triton-ascend:latest
docker run -it --device=/dev/davinci0 triton-ascend:latest bash
```

## 1.4 验证安装

创建 `verify_install.py` 并运行：

```python
import torch
import torch_npu
import triton
import triton.language as tl

# 验证 torch_npu 可用
print(f"PyTorch version: {torch.__version__}")
print(f"torch_npu available: {torch.npu.is_available()}")
print(f"NPU device count: {torch.npu.device_count()}")

if torch.npu.is_available():
    device = torch.npu.current_device()
    print(f"Current NPU device: {device}")

    # 获取硬件核数信息
    import triton.runtime.driver as driver
    props = driver.active.utils.get_device_properties(device)
    print(f"AI Core 数量: {props['num_aicore']}")
    print(f"Vector Core 数量: {props['num_vectorcore']}")

# 验证 Triton-Ascend 可用
@triton.jit
def _verify_kernel(x_ptr, y_ptr, n, BLOCK_SIZE: tl.constexpr):
    pid = tl.program_id(0)
    offsets = pid * BLOCK_SIZE + tl.arange(0, BLOCK_SIZE)
    mask = offsets < n
    x = tl.load(x_ptr + offsets, mask=mask)
    tl.store(y_ptr + offsets, x + 1.0, mask=mask)

# 运行验证
x = torch.randn(1024, device="npu")
y = torch.empty_like(x)
grid = (triton.cdiv(1024, 256),)
_verify_kernel[grid](x, y, 1024, BLOCK_SIZE=256)

diff = torch.max(torch.abs(y - (x + 1.0))).item()
print(f"Triton kernel 验证: {'PASS' if diff < 1e-5 else 'FAIL'} (max diff: {diff})")
print("\n所有组件安装验证通过!")
```

```bash
python verify_install.py
```

预期输出：
```
PyTorch version: 2.7.1+cpu
torch_npu available: True
NPU device count: 1
AI Core 数量: 24
Vector Core 数量: 48
Triton kernel 验证: PASS (max diff: 0.0)

所有组件安装验证通过!
```

## 1.5 常见安装问题

### 问题 1：`No matching distribution found for torch==2.7.1+cpu`

```bash
# 先手动安装 torch，再安装 torch_npu
pip install torch==2.7.1+cpu --index-url https://download.pytorch.org/whl/cpu
pip install torch_npu==2.7.1
```

### 问题 2：`torch.npu.is_available()` 返回 False

- 检查 CANN 是否正确安装：`which npu-smi`
- 检查环境变量：`echo $ASCEND_HOME_PATH`
- 检查 NPU 驱动：`npu-smi info`

### 问题 3：CANN 版本不兼容

Triton-Ascend 3.2.0 需要 CANN 8.5.0，请确认版本匹配。
